# 2D Drone Simulator in TypeScript

TypeScript と HTML5 Canvas で作られた、シンプルな2Dドローン操縦シミュレーターです。
モダンで超高速なフロントエンドビルドツール「Vite」を使用して構築されています。

## 特徴

- **物理演算ベースの挙動**: 重力、空気抵抗、そして機体の傾きに応じた推力（スラスト）のベクトル分解をリアルタイムに計算しています。
- **シンプルな操作性**: キーボードの矢印キーまたはWASDキーで直感的に操縦可能。
- **軽量・高速**: 外部の重量級ゲームエンジンを使わず、ピュアなTypeScriptとCanvas APIのみで実装しています。

## 操作方法

| キー | アクション |
| :--- | :--- |
| **`↑`** または **`W`** | **メインスラスト（推力）上昇** / 上方向へ加速 |
| **`←`** または **`A`** | **機体を左に傾ける**（左への推進力が生まれます） |
| **`→`** または **`D`** | **機体を右に傾ける**（右への推進力が生まれます） |

> 💡 **操縦のコツ:** ドローンは真上にしか推力を出せません。左右に移動したいときは、まず機体をその方向に少し傾けてから、推力（`↑`）を入れてみてください。

## 必要要件

- Node.js (v18 以上推奨)
- npm (Node.js に付属)

## セットアップと起動方法

### 1. 依存ライブラリのインストール
プロジェクトのルートディレクトリで以下のコマンドを実行し、TypeScriptとViteをインストールします。

```bash
npm install
2. 開発サーバーの起動
ローカルの開発サーバーを立ち上げます。

Bash
npm run dev
起動後、ターミナルに表示されるURL（例: http://localhost:5173/）にブラウザでアクセスすると、シミュレーターをプレイできます。

3. プロダクションビルド
本番公開用に最適化されたJavaScriptファイルを書き出す場合は、以下のコマンドを実行します。dist フォルダに公開用ファイルが生成されます。

Bash
npm run build
フォルダ構成
Plaintext
├── index.html          # 画面の土台となるHTML（Canvasを配置）
├── package.json        # プロジェクトの設定・依存ライブラリ管理
├── tsconfig.json       # TypeScriptのコンパイル設定
└── src/
    ├── drone.ts        # ドローンの物理演算・衝突判定ロジック
    └── main.ts         # キーボード監視、ゲームループ、Canvas描画処理
